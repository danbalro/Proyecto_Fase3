Introduccion:


La siguiente pagina fue desarrollada por Daniel Steeven Balanta y Rolando Barrios estudiantes de la Universidad Nacional Abierta y a Distancia, quienes cursan el programa de Ingenieria en Sistemas. La informacion encontrada es de autoria propia y busca informar los metodos y procesos para enlazar una pagina con codigo HPH hacia una base de datos (MySQL), el contenido se compone de lecturas, videos, imagenes y juegos que ayudan a comprender las tematicas.

Objetivos del Curso:


Crar un repositorio en la aplicacion GitHub con el fin de realizar el desarrollo de una pagina WEB que contenga lenguaje HTML y CSS, conocer el codigo de programacion y desarrollar metodos y practicas de programacion.
Explicar la tematica seleccionada por el grupo mediante el contenido que se exponga en la pagina web, en este caso la tematica es: CONEXION DE UNA PAGINA CON LENGUAJE PHP HACIA BASE DE DATOS SQL.
